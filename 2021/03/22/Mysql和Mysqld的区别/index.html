Mysql和Mysqld的区别

2021-03-22

mysql 是个命令行程序。

mysqld 是服务。

linux 系统里一般的服务都是以 d 结尾的，比如 httpd，vsftpd 等等。

d 的全拼应该是 daemon，也就是守护程序的意思，常驻于后台。
