MariaDB二进制安装

2021-03-29 | 数据库

1.下载MariaDB的安装文件
https://downloads.mariadb.org/mariadb/+releases/

2.把安装文件上传至/tmp

3.解压
tar -zxvf mariadb-10.5.8-linux-x86_64.tar.gz

4.将解压出来的文件复制到 /usr/local/ 目录下
mv mariadb-10.5.8-linux-x86_64 /usr/local/

5.修改目录名
cd /usr/local/
mv mariadb-10.5.8-linux-x86_64/ ./mariadb

6.创建用户和组，将mysql放入mysql组
sudo adduser mysql
groupadd mysql
usermod -G mysql mysql

7.安装
cd /usr/local/mariadb
sudo ./scripts/mariadb-install-db --user=mysql
chown -R root .
chown -R mysql data
./bin/mysqld_safe --datadir='./data'
cp support-files/mysql.server /etc/init.d/
chmod +x /etc/init.d/mysql.server
update-rc.d mysql.server defaults
ln -s /usr/local/mariadb/bin/mysql /usr/bin/
ln -s /usr/local/mariadb/bin/mysqld_safe /usr/bin
ln -s /usr/local/mariadb/bin/my_print_defaults /usr/bin

vim /etc/init.d/mysql.server
basedir=/usr/local/mariadb
datadir=/usr/local/mariadb/data

8.启动数据库
service mysql start
service mysql status

9.修改密码
service mysql stop
sudo mysqld_safe --skip-grant-tables --skip-ntworking &
